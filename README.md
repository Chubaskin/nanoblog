# nanoblog
Generador de contenido HTML offline

## Descripción
Lee su contenido desde un repositorio simple y las convierte en páginas con HTML estático
Ponemos de este modo la carga de generar contenido en el servidor y generamos páginas livianas para ser desplegadas en el cliente.

### Primera versión
- Desconectada
Corre en una máquina "apagada", de modo que se activa de forma regular, lee el archivo de contenido y se vuelve a desactivar
- JSon
El archivo con el contenido se encuentra en un repositorio json que no necesariamente se encuentra en la máquina.
